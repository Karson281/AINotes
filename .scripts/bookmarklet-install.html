Save to Vault

方法一：快捷鍵（最簡單）

在任何 AI 頁面，㩒書籤 Save to Vault，然後喺 Obsidian 貼上：

1. 加 Bookmarklet 去書籤欄：

Save to Vault

拖上面掣去書籤欄

2. 㩒書籤 → 全選 → Ctrl+C 複製

3. Obsidian 開新 Note → Ctrl+V 貼上

方法二：Python CLI（已可用）

用 vault-tool.py import 功能：

python3 D:/ObsidianVault/AINotes/.scripts/vault-tool.py import <你的檔案.md>

或者將下載咗嘅 .md 直接搬去：

D:\ObsidianVault\AINotes\1-AI-Conversations\
