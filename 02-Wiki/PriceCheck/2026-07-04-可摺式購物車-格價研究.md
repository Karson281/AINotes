## 📝 揀選建議（放車尾用）

### ✅ 必 check 位

```
□ 折疊後 size → 放得入車尾就算
   → LON10 / Livehon 摺疊後 A4 大小 ✅

□ 車輪類型 → 4輪萬向 vs 2輪爬梯
   → 放車尾用 4輪萬向 最方便 ✅

□ 承重能力 → 一般 10-20kg 夠用
   → LON10 50L 容量足夠一星期買餸

□ 重量 → 車尾用唔使太輕，但要單手拎到
   → 1-2kg 最理想
```

### ❌ 唔適合放車尾嘅款

- 助行購物車（Ziglite WBZ/WBU）— 太大件
- 帶坐椅款式 — 佔位
- 硬殼款式 — 摺疊後仍然大舊
