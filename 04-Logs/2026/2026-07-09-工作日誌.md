# 工作日誌 — 2026-07-09

## 完成事項

### 1. Telegram Bot x DeepSeek AI Integration 🎉

n8n Webhook + Polling Container + DeepSeek AI 三合一 Telegram Bot MVP 正式落成。

#### Architecture
```
Telegram User ↔ Poller Container ↔ n8n Webhook → Code node → DeepSeek API → Telegram Reply
```

#### 技術突破（SQLite 架構，非 PostgreSQL）
市面上 n8n Telegram Bot 教學全部假設 PostgreSQL + 公開 webhook。呢套 **SQLite + polling container + DB direct surgery** 嘅做法係少有嘅成功案例。

### 2. 關鍵 Bug 排查

| 1 | Bot 只 Echo 唔回答 | DeepSeek 未接入 | 加 Code node call DeepSeek API |
