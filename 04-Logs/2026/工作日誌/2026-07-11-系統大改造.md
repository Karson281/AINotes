# 2026-07-11 系統大改造：n8n/Node-RED → Hermes Script-Driven

## 背景
用戶原本用 n8n + Node-RED + Telegram Bot 處理股票指令，系統複雜、維護成本高、多處 hardcoded secrets。

## 目標
- 移除所有 workflow 引擎（n8n/Node-RED）
- 轉為 pure script-driven，由 Hermes Agent 直接處理
- 零 hardcoded 憑證
- 輕量化系統

## 已完成工作

### 1️⃣ 架構遷移：Workflow → Script
- 建立 6 個指令 scripts：`stock-command.py` / `portfolio-command.py` / `dividend-command.py` / `watchlist-command.py` / `summary-command.py` / `help-command.py`
- 全部經 Hermes Assistant 直接執行，唔需要 / 前綴
- 零 Docker overhead

### 2️⃣ Stock Command 強化
- 加入股票分類系統（防守/增長/週期/收益/價值）
- 加入即時基本因素（PE、股息率、Beta、52週範圍、分析師目標價）
- 加入按股票類型調整嘅風險提示
- 防守股 RSI 30-40 唔等於要減倉
- 14行 code → 230行完整分析
