# OneDrive Backup Setup

## 點樣用 OneDrive backup vault（唔使搬檔案）

### Step 1: 開 OneDrive Settings
- 右下角工作列 → 右鍵 OneDrive icon → **Settings**
- 或者：開始 Menu → **OneDrive** → **Settings**

### Step 2: 加 D: 盤 backup
- 揀 **「Sync and backup」** → **「Manage backup」**
- 㩒 **「Add a folder」**
- 揀 `D:\ObsidianVault\AINotes`
- 確認

### 結果
```
你 save 對話 → vault 自動 backup 上 OneDrive
→ 手機/其他電腦 OneDrive 見到晒所有 Note
```

### 注意
- 第一次 sync 要等一陣（睇檔案數量）
- 之後每次改檔案會自動 sync
- 唔影響 Obsidian、Watcher、Extension 任何功能
