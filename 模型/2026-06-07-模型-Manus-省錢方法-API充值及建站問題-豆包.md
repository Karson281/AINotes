一、整體協作邏輯（AI 爬蟲標準流程）
Tavily 負責「找網址」，自寫爬蟲 / LLM 負責「扒內容 + AI 清洗數據」，分 3 步：
Tavily Search：關鍵詞批量搜網站，篩高相關 URL（自動過濾垃圾鏈接、防封域名）
分流：優先用 Tavily Extract 批量扒頁面；難爬網站（JS 渲染、反爬）交自寫 Python 爬蟲（Selenium/Requests）
AI 後處理：大模型（GPT/DeepSeek）清洗爬取原文、分類、結構化存庫
Tavily 三大 API 對應爬蟲場景
表格
API 函數	用途（爬蟲角色）

三、分段代碼實現：Tavily 搜 URL + 雙爬蟲（官方提取 + 自寫爬蟲）
步驟 1：Tavily 關鍵詞搜尋，輸出有效網址
python
import os
from dotenv import load_dotenv
from tavily import TavilyClient
import requests
from bs4 import BeautifulSoup

load_dotenv()
client = TavilyClient(api_key=os.getenv("TAVILY_KEY"))

# 1. Tavily搜網址，advanced深度搜尋，返回10條結果
res = client.search(
    query="香港電子產品市場最新報價",
    search_depth="advanced",
    max_results=10,
    include_raw_content=False # 暫唔要原文，淨拎url
)
# 拎所有搜索出來嘅網址
url_list = [item["url"] for item in res["results"]]
print("Tavily搜到URL：",url_list)

步驟 2：方案 A｜Tavily 官方批量提取（優先使用，免反爬、自動去廣告）
python
# Tavily批量扒正文（一次最多20個URL）
extract_data = client.extract(urls=url_list[:8])
# 輸出乾淨內容
for page in extract_data["results"]:
    print(f"\n【{page['url']}】\n正文：{page['raw_content'][:500]}")

步驟 3：方案 B｜自寫 AI 爬蟲（Tavily 提取失敗嘅網站手動爬）
python
def custom_spider(target_url):
    """自訂爬蟲：Tavily搞唔掂嘅JS/反爬頁面使用"""
    headers = {"User-Agent":"Mozilla/5.0"}
    try:
        resp = requests.get(target_url,headers=headers,timeout=10)
        soup = BeautifulSoup(resp.text,"html.parser")
        content = soup.get_text(strip=True,separator=" ")
        return content
    except Exception as e:
        return f"爬取失敗：{str(e)}"

# 剩餘URL用自寫爬蟲
for bad_url in url_list[8:]:
    text = custom_spider(bad_url)
    print("自爬內容：",text[:300])
