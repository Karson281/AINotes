## 第一次使用

### 1. 建立每日筆記
用 Obsidian 嘅 Daily Notes 插件（Ctrl/Cmd + Shift + D）自動建立今日筆記。

### 2. 記錄 AI 對話
- 複製對話內容
- 用 Templates/AI-Conversation.md 模板建立新 Note
- 放 `1-AI-Conversations/` 目錄
- 喺 frontmatter 填 `source`（邊個 AI 平台）同 `tags`

### 3. 用 Proma Agent
叫 Proma Agent 整理筆記時，佢會：
- 直接讀寫 vault 入面嘅 .md 檔案
- 自動分類、加標籤、建立關聯
- 生成每週回顧同洞察

### 4. 每週回顧
每個星期五用 Templates/Weekly-Review.md 做一週總結。
