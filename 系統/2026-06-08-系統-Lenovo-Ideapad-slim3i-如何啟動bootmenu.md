🔑 啟動 Boot Menu 的方法

完全關機

確保電腦不是休眠或待機，而是完全關閉。

開機並立即按鍵

按下電源鍵後，當 Lenovo 標誌出現時，立刻快速連續按 F12。

若鍵盤需要功能鍵組合，則使用 Fn+F12。

選擇開機裝置

進入 Boot Menu 後，可用方向鍵選擇：

內建硬碟

USB 隨身碟

光碟機（若有）

網路開機 (PXE)

進入 BIOS（如需更改設定）

若要進入 BIOS 而非 Boot Menu，則在 Logo 畫面時按 F1。

部分 Ideapad 機型也可透過 Novo Button（機身側邊的小孔按鈕）進入 BIOS/Boot Menu。

⚠️ 注意事項

時間掌握很重要：必須在 Logo 畫面出現時立即按鍵，錯過就要重新開機再試。

BIOS 設定影響選單：若某裝置在 BIOS 中被停用，即使在 Boot Menu 也不會顯示。

安全開機 (Secure Boot)：若要從 USB 開機，可能需要在 BIOS 中關閉 Secure Boot。

對，這樣的操作方式是正確的思路 👍。
在 Lenovo Ideapad Slim 3i 上，Boot Menu 的快捷鍵通常是 F12，但因為很多 Ideapad 機型的功能鍵預設是「多媒體模式」，所以必須 同時按住 Fn 再狂按 F12 才能觸發 Boot Menu。

建議操作方法：

完全關機 → 確保不是休眠或待機。

按電源鍵開機 → 畫面出現 Lenovo 標誌時，立刻：

一隻手按住 Fn，

另一隻手快速連續按 F12。

Boot Menu 出現 → 用方向鍵選擇 USB、硬碟或其他開機裝置。

小技巧：

如果你覺得 Fn+F12 太麻煩，可以在 BIOS 裡把 Hotkey Mode 關掉，這樣以後只要單獨按 F12 就能進入 Boot Menu。

若還是進不去，可以試試 Novo Button（機身側邊的小孔，用迴紋針按下去），它會直接給你 BIOS/Boot Menu 選項。
