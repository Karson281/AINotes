以下係一套
標準 workflow
，專為香港小六中英數試卷設計，每次用 AI 快速出新卷（10-20 分鐘內搞掂）。呢個 workflow 結合Monsha（worksheet ＋圖片生成）、ChatGPT/Claude（題目改寫）、Canva（版面排版），唔使寫程式，純 no-code。

準備工作（每次 2 分鐘）

掃描／影低你手上嘅小六試卷（PDF 或多張 JPG）。

用 Google Lens 或 ChatGPT Vision 快速分析原卷考點（見 Prompt 1）。

記低核心元素：

科目：中/英/數

考點：例如數學「分數、幾何」；中文「閱讀理解、造句」

題型：MCQ、填空、開放題、圖表題

難度：P6 標準

總題數、時間、分數

步驟 1：用 ChatGPT Vision 抽取考點（3 分鐘）

Prompt 1（直接 copy paste，上載原卷圖片）：

text
分析呢份香港小六[科目，例如：數學]試卷。  
1. 列出所有考點同子題型（例如：分數加減法、面積計算）。  
2. 每題類型比例（MCQ 幾多%、填空幾多%）。  
3. 有幾多圖片題？描述圖片類型（例如：圓形圖表、幾何圖）。  
4. 總分、分鐘數、版面結構（Part A/B、總題數）。  
5. 輸出 JSON 格式，方便 copy。

輸出：一張 JSON 表，儲低做「藍圖」。

步驟 2：用 Monsha 生成新題目（5-8 分鐘）

去 monsha.ai，登入免費版（有 daily limit，夠晒用）。
Prompt 2（輸入你 JSON 考點，按科目調整）：

text
Grade: Hong Kong Primary 6  
Subject: [數學/中文/英文]  
Language: [繁體中文 / English / Bilingual]  
Create a worksheet with:  
- Total questions: [原卷總數，例如 30 題]  
- Question types: [從 JSON copy，例如 40% MCQ, 30% fill-in, 30% open with diagrams]  
- Topics: [JSON 考點，例如 Fractions addition, Geometry shapes, Reading comprehension]  
- Difficulty: P6 standard  
- Include diagrams/images for [圖片題，例如 bar charts, triangles]  
- Format: Exam paper style with sections (Part A, Part B), total marks [原卷總分], time [分鐘]  
- Make questions ORIGINAL, similar difficulty to HK Primary 6 TSA exams.

輸出：即時生成 worksheet PDF，帶圖片同版面！直接 download。

步驟 3：微調同加答案（3 分鐘）

如果題目唔夠「香港味」，copy Monsha 題目回 ChatGPT，用 Prompt 3 改寫：

text
呢啲題目來自 Monsha，改寫成全新香港小六[科目]題目：  
1. 保持相同考點同難度。  
2. 改數字、情境、選項，令佢完全唔同原卷。  
3. 加答案 key 同解釋。  
[貼入 Monsha 題目]

儲答案 key 做 reference。

步驟 4：用 Canva 完美版面（3-5 分鐘）

去 Canva.com，search "exam paper template" 或用你原卷做 background。

Copy Monsha 題目／圖片，貼入 template。

調整 font（用 Arial 或標楷體）、layout，跟原卷一模一樣。

Export PDF，高解像度印出。
