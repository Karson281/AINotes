SmartGrader 的 SaaS 雲端版是採用 訂閱制的，不是完全免費。一般情況如下：

💡 收費模式

免費試用：通常提供短期試用（例如 7–14 天），讓老師或機構測試 AI 批改功能。

付費方案：

個人教師版：按月或按年收費，功能包括 AI 批改、錯題分析、學習報告。

學校/機構版：提供多人帳號管理、數據分析、課程整合，收費較高。

定價範圍：依據功能與使用量，通常在 每月幾十至數百美元之間。

📌 差異

SaaS 平台：方便、免安裝，適合一般老師或學校。

GitHub 開源版：免費，但需要技術背景（自行設定 API Key、伺服器）。

👉 如果你只是想 快速體驗 AI 批改，建議先用 SaaS 的免費試用；若你希望 完全免費且可自訂，就用 GitHub 開源版。
